# joelib

A java library for me.

# Installation

### Gradle

Add dependency to your **build.gradle**

```java
repositories {
    jcenter()
    maven { url "https://jitpack.io" }
}
dependencies {
    implementation 'com.github.kcwongjoe:joelib:v0.0.3'
}
```
